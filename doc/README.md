# 文档目录

本目录包含项目的详细文档。

## 文档列表

### 使用文档
- **[USAGE.md](USAGE.md)** - 详细的使用指南，包括快速开始、常见问题、性能优化等

### 项目文档
- **[STRUCTURE.md](STRUCTURE.md)** - 项目结构详细说明，包括目录结构、模块说明等
- **[PROJECT_SUMMARY.md](PROJECT_SUMMARY.md)** - 项目完成情况总结，功能列表、技术栈等
- **[FILES.md](FILES.md)** - 项目文件清单，所有文件的说明

### 开发文档
- **[REFACTORING.md](REFACTORING.md)** - 项目重构说明，改进对比、迁移指南等
- **[TROUBLESHOOTING.md](TROUBLESHOOTING.md)** - 故障排除指南，常见问题和解决方案

### 课程设计文档
- **[aim.md](aim.md)** - 课程设计要求

## 快速导航

- **新手入门**: 先阅读 [USAGE.md](USAGE.md)
- **了解项目**: 查看 [STRUCTURE.md](STRUCTURE.md) 和 [PROJECT_SUMMARY.md](PROJECT_SUMMARY.md)
- **遇到问题**: 参考 [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
- **开发扩展**: 阅读 [REFACTORING.md](REFACTORING.md)

## 文档维护

所有文档使用 Markdown 格式编写，建议使用支持 Markdown 的编辑器查看。

如需更新文档，请确保：
1. 保持格式一致
2. 更新相关链接
3. 更新目录索引（本文件）
